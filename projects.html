Below you can find a list of projects I coded from scratch.

Weather App

This Weather App is a simple and user-friendly web application that provides current weather information and a forecast for a specified city. The interface features a clean and responsive design built with Bootstrap, allowing users to search for a city and view detailed weather data. It uses open-source data and connects to OpenWeather via an API request. This app is hosted on Netlify and the code is available on Github.

World Clock Project

The World Clock is a dynamic web page that allows users to view the current time and date in various cities around the world. The interface features a select dropdown menu where users can choose a specific timezone or their current location, which the page is able to guess. The time and date information are automatically updated every second using the Moment.js library.
